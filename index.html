BTooltip

The BTooltip plugin JS,
created for destroy
tooltip bootstrap automatically!

§ Examples

# HTML

<button id="btnLeft" type="button" class="btn btn-default" data-toggle="tooltip" >Tooltip on left</button>

# Javascript

$('#btnLeft').click(function() {
  $('#btnLeft').btooltip({
    title : 'Tooltip on left',
    placement: 'left'
    destroy : true,
    time : 4000
  });
});
